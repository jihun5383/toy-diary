# Toy Diary

A modern, Tailwind-styled diary MVP built with Vite + React. Entries are saved locally in your browser (no backend required) with filters for date and keyword search.

## Features
- Create, edit, and delete diary entries with mood tagging
- Instant autosave to `localStorage`
- Date and keyword filtering with a modern, glassmorphism-inspired UI

## Getting started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production build:
   ```bash
   npm run preview
   ```

## Linting
